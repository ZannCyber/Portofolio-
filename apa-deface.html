APA ITU DEFACE/DEFACER

Deface adalah tindakan mengubah tampilan situs web secara tidak sah, yang merupakan salah satu bentuk kejahatan siber. Deface juga dikenal sebagai web defacement.

Bagaimana sihh serangan deface ituu? serangan deface ini bisa dilakukan dengan mengubah tampilan website seseorang tanpa izin atau ilegal, contoh nyaa website dpr yang tadinya pakai logo dpr atau resmi nya diganti dengan logo sihh defacer ituu sendiri, jadi intinya serangan deface ituu semacam mengubah atau mengedit halaman situs tanpa izin dan ilegal, kegiatan ini juga memiliki hukum bagi yang melakukan kegiatan ini seperti diindonesia akan terkena beberapa pasal, seperti pasal 30 UU ITE (mengakses sistem elektronik tanpa izin) hukuman nya maksimal 6 tahun penjara dan/atau denda maksimal Rp 600 juta.

bagaimana sih cara melakukan deface itu? serangan deface memiliki berbagai banyak teknik seperti sql injection,brute force,xss, dll. serangan yang popular atau mudah adalah sql injection dengan mencoba bypass halaman login ke web yang rentan terhadap sql injection, dengan menggunakan payload sql '=''or' untuk mengakses halaman admin atau dashboard dan lalu mengedit hingga mengupload file untuk merubah halaman tersebut.
